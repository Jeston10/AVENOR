# Background music for Avenor

Place your **soothing, calm, peaceful, and professional** audio files here. The app expects these filenames (or update `src/data/playlist.ts` to match your files):

| File name           | Suggested style / keywords                    |
|---------------------|-----------------------------------------------|
| `ambient-calm.mp3`  | Ambient, pads, soft, minimal                  |
| `soft-piano.mp3`    | Piano, gentle, emotional                      |
| `peaceful-waters.mp3` | Nature, water, meditation                   |
| `gentle-keys.mp3`   | Keys, light, uplifting                        |
| `quiet-space.mp3`   | Space, atmospheric, calm                     |
| `morning-light.mp3` | Acoustic, hopeful, warm                       |
| `stillness.mp3`     | Drone, meditation, stillness                  |
| `reflection.mp3`    | Reflective, contemplative                     |

## Royalty-free sources (for commercial use)

- **Pixabay** – https://pixabay.com/music/ (filter: Calm, Ambient, Corporate)
- **YouTube Audio Library** – https://studio.youtube.com → Audio Library (free for use)
- **Free Music Archive** – https://freemusicarchive.org (check each license)
- ** incompetech** – https://incompetech.com/music (Creative Commons, attribution often required)
- **Bensound** – https://www.bensound.com (royalty-free with attribution)

Download tracks that match the mood, rename them to the table above (or edit `web/src/data/playlist.ts`), and place them in this folder. The app will play them in order and loop the playlist.
